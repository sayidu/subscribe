# Subscribe

This is a quick one page subscription application with only two models: `Subscription` and `Plans`

## Setup
* Setup Database by running the following commands:
  * `rake db:setup`
  * `rake db:migration`
  * `rake db:seed`

* Run `bundle install`

* Start server with  `bundle exec rails s`
